Website Workshops

Helped organize a website workshop to teach NKU students how to design their own website.

read more...

We taught the basics of HTML and CSS, and showed them how to edit templates and use a network drive.

I designed six templates for students to fill in with their information.

I wrote a guide on accessing the drive and on the day of the event helped with instructing students.

The template site used JavaScript code I wrote to combine the HTML and CSS into one file, and allow students to change CSS on the fly.
